# One-shot answers: no Back navigation during the test

The design handoff draws a "← Back" button on the question screen (2b), but we deliberately drop
it: answers are one-shot commitments, in the Gallup "answer on instinct" spirit the 30-second soft
timer exists to create. Timer expiry commits the slider as it stands (untouched slider = the
centered Neutral), so every item always has exactly one answer recorded on first exposure and the
scorer never sees gaps or revisions. If you see the Back button missing relative to the wireframes,
it is not an omission — do not add it back without revisiting this decision.
